Política de Privacidade

Esta Política de Privacidade descreve como os dados são tratados no aplicativo Vendix.

1. Informações Coletadas

O aplicativo armazena os dados inseridos pelo próprio usuário, como produtos, clientes, vendas, forma de pagamento, configurações de caixa, dados de conta e demais informações necessárias para o funcionamento do sistema.

2. Finalidade do Tratamento

As informações são utilizadas para operação do PDV, controle de estoque, emissão de relatórios, gerenciamento de caixa, compartilhamento de cupom e restauração por backup.

3. Armazenamento Local

Os dados são mantidos localmente no dispositivo do usuário. O app não depende de armazenamento em nuvem para funcionar, salvo quando o próprio usuário optar por exportar ou compartilhar informações.

4. Compartilhamento

O Vendix não compartilha automaticamente os dados com terceiros. Qualquer compartilhamento ocorre somente por ação direta do usuário, como envio de vendas, relatórios ou backup.

5. Segurança

O aplicativo adota boas práticas para reduzir riscos de perda de dados, mas a segurança final também depende do cuidado do usuário com o dispositivo, backups e permissões concedidas.

6. Retenção e Exclusão

Os dados permanecem no dispositivo até que o usuário os remova, restaure um backup ou desinstale o aplicativo, observadas as limitações do sistema operacional.

7. Direitos do Usuário

O usuário pode consultar, editar, remover e restaurar os dados dentro dos recursos disponíveis no aplicativo.

8. Atualizações desta Política

Esta Política pode ser atualizada a qualquer momento para refletir melhorias do aplicativo, ajustes técnicos ou exigências legais.
