# Vendix - Política de Privacidade

Este repositório contém a página de **Política de Privacidade** do aplicativo **Vendix**, um sistema de PDV (Ponto de Venda) e gestão de estoque. 

A página foi desenvolvida em HTML5 e CSS3, sendo uma versão otimizada e responsiva do componente original criado em Flutter.

## 🛠️ Tecnologias

- **HTML5**
- **CSS3** (Variáveis CSS e Media Queries)
